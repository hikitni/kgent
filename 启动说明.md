# KgentV3 启动说明

> **KgentV3** — 本地文件变更监控 + AI 日报生成工具
> 版本：V3 | 平台：Windows 10/11

---

## 一、包内文件说明

| 文件 | 说明 |
|---|---|
| `KgentV3.exe` | 主程序（已打包，无需安装 Python） |
| `config.example.json` | 配置模板，需复制为 `config.json` 后编辑 |
| `ai_report_prompt.md` | AI 日报生成使用的提示词模板（可自定义） |
| `start-web.bat` | 双击启动 Web 管理界面 |
| `start-cli.bat` | 双击启动命令行交互菜单 |

---

## 二、首次使用（3 步上手）

### 步骤 1：复制并编辑配置文件

将 `config.example.json` 复制一份，重命名为 `config.json`，然后用记事本打开编辑：

```json
{
  "watch_paths": ["D:/你的项目路径"],
  "ai_provider": "openai",
  "ai_model": "deepseek-chat",
  "ai_api_key": "你的API密钥",
  "ai_base_url": "https://api.deepseek.com/v1"
}
```

**关键配置项：**

| 配置项 | 说明 | 示例 |
|---|---|---|
| `watch_paths` | 要监控的目录（支持多个） | `["D:/work/project-a", "D:/work/project-b"]` |
| `ignore_dirs` | 需忽略的目录名 | `[".git", "node_modules", "__pycache__"]` |
| `ignore_suffixes` | 需忽略的文件后缀 | `[".log", ".tmp", ".pyc"]` |
| `snapshot_root` | 快照数据存放目录 | `.daily_reporter`（默认） |
| `output_root` | 日报输出目录 | `reports`（默认） |
| `ai_provider` | AI 提供商 | `disabled` / `openai` / `zhipuai` / `ollama` |
| `ai_model` | 模型名称 | `deepseek-chat` / `glm-4` / `gpt-4o` |
| `ai_api_key` | API 密钥 | 从对应平台获取 |
| `ai_base_url` | API 地址（部分服务商需要） | `https://api.deepseek.com/v1` |

> **不需要 AI 功能？** 将 `ai_provider` 设为 `"disabled"` 即可，其他 AI 字段可留空。

### 步骤 2：启动程序

**方式 A —— Web 界面（推荐）：**

双击 `start-web.bat`，浏览器会自动打开 `http://localhost:7421`。

**方式 B —— 命令行菜单：**

双击 `start-cli.bat`，按照终端数字菜单操作。

### 步骤 3：开始使用

1. **打快照**：上班时点击「创建快照」记录初始状态
2. **继续工作**：正常编码，工具不会干扰你
3. **再打快照**：下班前再次创建快照
4. **生成日报**：选择两个快照对比，点击「生成日报」
5. **AI 美化**（可选）：对原始日报点击「生成 AI 日报」，等待异步完成

## 五、常见问题

### Q: 启动时提示"Windows 已保护你的电脑"？
A: 点击「更多信息」→「仍要运行」。这是因为 exe 没有数字签名，属于正常现象。

### Q: 端口 7421 被占用？
A: 使用命令行指定其他端口：
```
KgentV3.exe --web --port 8080
```

### Q: 如何在其他盘监控文件？
A: 编辑 `config.json` 中的 `watch_paths`，可添加多个绝对路径：
```json
"watch_paths": ["D:/project-a", "E:/project-b"]
```

### Q: 日报生成在哪里？
A: 默认在程序所在目录的 `reports/` 文件夹下，文件名格式为 `report-时间段.md`。

### Q: AI 日报生成很慢？
A: AI 日报已采用异步后台生成机制，提交后可继续操作页面，状态栏会实时显示生成进度。

## 六、命令行高级用法

```powershell
# 直接打一次快照（用于计划任务）
KgentV3.exe snapshot

# 带标签打快照
KgentV3.exe snapshot --label morning

# 指定配置文件
KgentV3.exe -c D:\work\my-config.json --web

# 指定端口且不自动打开浏览器
KgentV3.exe --web --port 8080 --no-browser
```
